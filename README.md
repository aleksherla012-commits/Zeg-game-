Labyrinth: Veil of Riddles to przeglądarkowa gra labiryntowa wykonana w technologii HTML5 Canvas oraz JavaScript. Gracz wciela się w bohatera przemierzającego tajemnicze labirynty pełne zagadek, pułapek i przeciwników. Celem gry jest odnalezienie wyjścia z każdego poziomu poprzez eksplorację mapy, zbieranie przedmiotów oraz rozwiązywanie prostych zagadek logicznych.

Gra oferuje:

płynne sterowanie klawiaturą,
system zdrowia (HP),
kilka poziomów o rosnącym poziomie trudności,
przeciwników patrolujących wybrane obszary mapy,
pułapki utrudniające rozgrywkę,
przedmioty do zbierania, np. klucze i mikstury,
styl graficzny pixel art,
prosty i czytelny interfejs użytkownika.

Projekt został wykonany przy użyciu:

HTML5,
CSS3,
JavaScript,
Canvas API.

W projekcie zastosowano mechanizmy takie jak system kolizji, pętla gry (game loop), animacje postaci oraz obsługa interakcji z obiektami na mapie. Gra działa bezpośrednio w przeglądarce i nie wymaga instalacji dodatkowego oprogramowania.

Celem projektu było stworzenie kompletnej gry 2D oraz rozwinięcie umiejętności programowania w JavaScript, projektowania poziomów i pracy zespołowej.
