# Labyrinth: Veil of Riddles
Labyrinth: Veil of Riddles to wciągająca przeglądarkowa gra labiryntowa stworzona w technologii HTML5 Canvas i JavaScript.  Bohater porusza się przez coraz trudniejsze labirynty, zbieraj przedmioty, unika pułapek i patrolujących przeciwników oraz rozwiązuje logiczne zagadki, aby przejść dalej. Gra oferuje płynne sterowanie klawiaturą, spójny styl pixel art, system zdrowia, kilka poziomów o rosnącym poziomie trudności oraz czytelny interfejs użytkownika – wszystko działa bezpośrednio w przeglądarce.
